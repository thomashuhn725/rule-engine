## An Experiment in Architecture Centered Development (ACD) As Applied to AI

This repository is the result of an experiment in generating code from design schematics instead of natural language. The goal was to see if more controllable, quality results could be obtained from an LLM by front-loading the process with proper design.
More can be read about this on Substack. https://thomashuhn.substack.com/p/ai-as-a-return-to-architecture-fundamentals.
